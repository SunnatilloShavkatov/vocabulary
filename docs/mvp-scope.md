# MVP Scope

## In-scope

### Admin (Flutter Web)

- Login (`email`, `password`)
- Vocabulary qo'shish
- Yangi admin qo'shish

### Oson Vocabulary client app (Flutter)

- Welcome page
- Vocabulary list page
- Search qismi

### Backend (Go)

- Gateway service
- Auth service
- Vocabulary service
- JWT asosida admin endpoint himoyasi

## Out-of-scope (MVPdan keyin)

- Social login
- Multi-language UI
- Offline sync
- Advanced analytics/dashboard
- Bulk import/export

## Functional acceptance (MVP)

1. Admin login qilib, so'z qo'sha olishi kerak.
2. Admin yangi admin qo'sha olishi kerak.
3. Oson Vocabulary client app so'zlar ro'yxatini ko'rsatishi kerak.
4. Oson Vocabulary client app qidiruv orqali natijani filtrlashi kerak.
5. Admin endpointlar token bo'lmasa `401` qaytarishi kerak.
